# Onboarding-Checkliste für neue Mitarbeiter

**Gültig ab:** Erster Arbeitstag  
**Verantwortlich:** HR + IT + Vorgesetzter

---

## Woche 1: Erste Schritte

### Tag 1: Willkommen

- [ ] Begrüßung durch Vorgesetzten
- [ ] Arbeitsplatz einrichten
- [ ] IT-Equipment erhalten (Laptop, Monitor, Zubehör)
- [ ] Zugangsdaten erhalten (E-Mail, VPN, Systeme)
- [ ] Unterschreiben:
  - [ ] Arbeitsvertrag
  - [ ] Datenschutzerklärung
  - [ ] IT-Sicherheitsrichtlinie
  - [ ] Vertraulichkeitsvereinbarung

### Tag 1-2: IT-Setup

- [ ] Laptop konfigurieren
- [ ] E-Mail-Konto einrichten
- [ ] VPN-Zugang testen
- [ ] Passwort-Manager installieren
- [ ] Multi-Faktor-Authentifizierung aktivieren
- [ ] Software installieren:
  - [ ] Office 365
  - [ ] Teams / Slack
  - [ ] Projekt-Management-Tool
  - [ ] Zeiterfassung

### Tag 2-3: Einarbeitung

- [ ] Unternehmensvorstellung
- [ ] Organigramm durchgehen
- [ ] Teammitglieder kennenlernen
- [ ] Pausenzeiten & Meeting-Kultur erklärt
- [ ] Erste Projekt-Dokumentation erhalten

### Tag 3-5: Erste Aufgaben

- [ ] Erstes "Hello World" Projekt
- [ ] Code-Review-Prozess erklärt
- [ ] Git-Workflow erklärt
- [ ] Entwicklungsumgebung einrichten

---

## Woche 2-4: Vertiefung

### IT & Security

- [ ] IT-Sicherheitsschulung absolviert
- [ ] Zugriffsrechte auf Projekt-Systeme
- [ ] Backup-Prozess erklärt
- [ ] Incident-Response-Verfahren geklärt

### Prozesse & Tools

- [ ] Projektmanagement-Tool eingeführt
- [ ] Dokumentationssystem erklärt
- [ ] Meeting-Rhythmen verinnerlicht
  - [ ] Daily Standup
  - [ ] Wochen-Meeting
  - [ ] Sprint Planning (falls agil)
- [ ] Abrechnungsprozess geklärt

### Fachliche Einarbeitung

- [ ] Erstes kleines Ticket bearbeitet
- [ ] Erster Pull Request erstellt
- [ ] Code Review durchgeführt
- [ ] Architektur-Überblick erhalten

---

## Monat 1-3: Integration

### Kommunikation & Kultur

- [ ] 1:1-Gespräche mit Vorgesetzten
- [ ] Feedback-Runde nach 4 Wochen
- [ ] Teamevent teilgenommen
- [ ] Mentoring-Programm gestartet (falls vorhanden)

### Weiterbildung

- [ ] Produkt-Schulung absolviert
- [ ] Externe Schulung geplant (Budget klären)
- [ ] Zertifizierung identifiziert

### Ziele

- [ ] 30-Tage-Ziele definiert
- [ ] 90-Tage-Ziele definiert
- [ ] Erste Selbsteinschätzung
